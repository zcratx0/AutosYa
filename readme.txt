Proyecto Automotora

Autor: Matías Sánchez.
Version: 0.2
Fecha de inicio: 26/03/2022
Fecha de fin: 10/04/2022

Este proyecto se basa en la creacion de un sistema de gestion para el alquiler de automoviles a travez de la WEB (De forma online).
Este proyecto cuenta con un sistema para la gestion (backend) y un sitio web para su utilización (frontend)
Este proyecto cuenta con las funciones necesarias para poder gestion un sistema de renta de automoviles.
El sitio web y el backend esta construido a base de PHP y MySQL, a su vez el sitio web incorpora HTML5, CSS, Javascript con los framework de Materialize (https://materializecss.com/) y Jquerry(https://jquery.com/)
El sistema corre en PHP 7.4.19, MySQL 5.7.33, Apache 2.4.47

Es compatible con php 7.0 en adelante. 
Es compatible con MySQL 5.0 en adelante.
Es compatible con MariaDB 10
